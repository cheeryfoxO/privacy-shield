🛡️ 隐私政策 — 隐私护盾

最后更新：2026 年 6 月 12 日

概述

隐私护盾是一款浏览器扩展，用于帮助用户了解当前网页的隐私状况。本隐私政策说明我们如何（以及不）收集、使用和处理数据。

核心承诺：所有数据处理均在用户本地浏览器中进行，不会向任何服务器发送任何数据。

我们不会收集的数据

  • ❌ 个人信息（姓名、邮箱、地址等）
  • ❌ 浏览历史记录
  • ❌ Cookie 的值或内容
  • ❌ 表单输入或密码
  • ❌ 网页正文内容
  • ❌ 用户 IP 地址
  • ❌ 设备标识符或指纹

本地处理的数据

以下数据仅在用户设备本地处理，绝不离开浏览器：

  • 当前页面 URL（仅用于提取域名，不存储或传输）
  • 页面加载的第三方资源域名
  • Cookie 元数据（名称、域名、类型 — 不读取 Cookie 值）
  • Canvas API 调用记录（方法名、画布尺寸）
  • WebRTC ICE Candidates（IP 地址 — 不存储或传输）

存储机制

  • 用户设置通过 chrome.storage.sync 在登录设备间同步
  • 扫描报告暂存于 chrome.storage.local，关闭标签页时自动清除
  • 无数据被发送到远程服务器

权限使用说明

权限用途
cookies读取并分析 Cookie 的域名、类型、过期时间，不读取值
webRequest观察网络请求以识别第三方域名，不拦截或修改请求
storage本地保存用户设置和临时扫描结果
tabs获取当前标签页 URL 以关联扫描结果
<all_urls>在用户访问的网页中注入隐私检测脚本

第三方服务

本扩展不集成任何第三方分析、广告、遥测或追踪服务。

数据安全

由于所有数据均在本地处理，不存在数据泄露风险。扩展代码完全开源，欢迎审查。

🇬🇧 Privacy Policy — English

Last Updated: June 12, 2026

Overview

Privacy Shield is a browser extension that helps users understand the privacy status of web pages they visit. All data processing happens exclusively on the user's device. No data is ever transmitted to any server.

Data We Do NOT Collect

  • ❌ No personal information (name, email, address, etc.)
  • ❌ No browsing history
  • ❌ No cookie values
  • ❌ No form data or passwords
  • ❌ No page content
  • ❌ No IP addresses
  • ❌ No device identifiers or fingerprints

Local Processing Only

All data is processed locally in the browser. No data leaves the user's device.

Permission Usage

PermissionPurpose
cookiesAnalyze cookie metadata (domain, type, expiry). Values are NEVER read.
webRequestObserve network requests to identify third-party domains. Requests are NEVER intercepted or modified.
storageSave user preferences and temporary scan results locally.
tabsAssociate scan results with the current tab.
<all_urls>Inject detection scripts into pages the user visits.

Third-Party Services

This extension integrates NO third-party analytics, advertising, telemetry, or tracking services.

Data Security

Since all data is processed locally, there is no risk of data breach. The source code is fully open and available for review.
